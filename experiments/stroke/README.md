# Stroke experiments

This directory contains the real-data stroke experiments from the paper.

Expected input data location:

- `../../data/stroke/raw/`

Main notebooks:

- `yeo7_GW_segmentation.ipynb`
- `data_reload_and_plot.ipynb`

Because the real dataset is not redistributed here, you may need to adjust the filename-definition cells in
`yeo7_GW_segmentation.ipynb` to match your local copies.
